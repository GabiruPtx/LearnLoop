## _📖 Como um(a) usuário(a) do sistema quero criar projetos para organizar minhas atividades_

## **🎯 Objetivo**

_Criar um projeto para iniciar o gerenciamento das atividades. _

## **✅ Critérios de Aceitação**
Esses são os requisitos mínimos que precisam estar presentes para que a história seja considerada "pronta":
- _Formulário mínimo com campos: Nome (obrigatório) e Descrição (opcional)_
- _Criador assume automaticamente a função de administrador_
- _Visibilidade imediata na dashboard do usuário_
- _Mecanismo pós-criação para adição de membros_
